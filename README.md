# Node.js Web Form
This is a user login and registration web form using Node.js, Express, MySQL, EJS and some other packages.

```sh
# How to set up
cd User-Login && ./setup.sh

# How to run server
cd User-Login && node main.js

# Open "http://localhost:3000/" in your browser

```
